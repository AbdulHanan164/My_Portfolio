# AbdulHanan164 Portfolio

Welcome to the source code for my personal portfolio!

## Project Structure

- **Frontend**: Contains HTML, CSS (`style.css`), and JavaScript (`main.js`) for the visual interface of the portfolio.
- **Backend (`/backend`)**: A FastAPI application that handles form submissions (like the Contact Me form) and sends emails securely using Gmail SMTP and a Google App Password.

## Backend Setup
For detailed instructions on running and deploying the FastAPI backend to Vercel, please see [backend/README.md](backend/README.md).

## Local Development (Frontend)
Simply open `index.html` in any web browser to view the portfolio.
